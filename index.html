This website is a project designed to display my ability as a front-end web designer.

The links above will take you to a calculator of Ohm's equation, and three different tests.

The calculator can take two inputs and return the missing variable in the equation.
The tests are prefilled with information with a "reveal" button that disappears to reveal the correct answer.
There are three tests, one for each variable in the Ohm's-law equation.

The color scheme, fonts, and "selected" buttons are designed to be a more 1990's industrial website style.

This page is fully responsive to different viewport sizes.
